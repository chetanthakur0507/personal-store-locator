# 🏪 Smart Store Item Locator

A **Full Stack Inventory Management System** that helps store staff instantly find the exact location of products inside large stores.

Instead of spending **5–10 minutes searching for items**, staff can now locate them in **just a few seconds**.

📍 **Item Location Structure**

```
Floor → Aisle → Rack → Shelf
```

This system improves **staff productivity, customer experience, and inventory management.**

---

# 🚀 Features

## 🔍 Smart Item Search

* Real-time item search
* Shows exact store location
* Displays stock quantity
* Low stock warnings

---

# 📦 Inventory Management

Admin can manage the entire store inventory.

* Add new items
* Edit existing items
* Delete items
* Category based organization
* Precise location mapping
* Stock quantity tracking

---

# 👤 Staff Dashboard

The staff dashboard is optimized for **fast store-floor operations**.

Staff can:

* Search items instantly
* View item location
* Sell items directly from the dashboard
* Select quantity for sales

When an item is sold:

* Inventory quantity updates automatically
* Admin dashboard updates instantly
* Sales record stores:

  * Item name
  * Quantity sold
  * Price
  * Time of sale

---

# 📈 Trending Items

Items that sell frequently automatically appear:

* In a **Trending section on the main website**
* At the **top of the staff dashboard**

This allows staff to quickly access popular products without searching repeatedly.

---

# 📊 Admin Dashboard

The admin panel provides powerful insights about store performance.

Admin can view:

* Total inventory
* Category distribution
* Low stock alerts
* Sales analytics
* Daily sales performance
* Weekly sales reports
* Total orders
* Revenue generated
* Top selling products
* Recent sales activity

Every sale made by staff **updates the admin dashboard in real-time.**

---

# 👥 Staff Account Management

Admin can create accounts for store staff.

Features:

* Create staff accounts
* Manage existing staff
* Secure login system
* Role-based permissions

---

# 🔐 Role Based Access Control

| Role         | Permissions                                                     |
| ------------ | ------------------------------------------------------          |
| Admin        | Full inventory management, analytics, staff management          |
| Staff        | Search items, sell items, update stock                          |
| Public Users | Can search item locations and see the price (no login required) |
